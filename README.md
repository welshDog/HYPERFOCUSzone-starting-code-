- **Python 3.9+** (Required for core systems)
- **Node.js 16+** (Required for web portals)
- **Docker** (Recommended for development)
- **Signed NDA** (Required for access)

### **Development Setup:**
```bash
# Clone the repository (authorized personnel only)
git clone https://github.com/welshDog/HYPERFOCUSzone-PRIVATE.git
cd HYPERFOCUSzone-PRIVATE

# Install dependencies

# Set up environment
```
